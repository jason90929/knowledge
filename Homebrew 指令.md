*搜尋應用程式：*
`$ brew cask search 應用程式名稱`

*安裝應用程式：*
`$ brew cask install 應用程式名稱`

*移除應用程式：*
`$ brew cask uninstall 應用程式名稱`

*搜尋指令：*
`$ brew search 指令名稱`

*安裝指令：*
`$ brew install 指令名稱`

*移除指令：*
`$ brew uninstall 指令名稱`

### **如何更新指令版本？**

*更新版本列表：*
`$ brew update`

*查看應用程式簡介：*
`$ brew info 指令名稱`

*取消使用指令：*
`$ brew unlink 指令名稱`

*安裝指令：*
`$ brew install 指令名稱`

*移除多餘舊版本：*
`$ brew cleanup 指令名稱`
